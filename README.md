# CS2-AA2.1
Stay In or Go Out : MBTI Tester

Myers-Briggs Type Indicator (MBTI) is a test where people get grouped into 16 personality types based on their tendency or what they choose to do in certain situations. These persosnality types are a combination of : Extroversion, Introversion, Sensing, Intuition, Feeling, Thinking, Prospecting, Judging. This program was created to inform users about their MBTI and habits they usually do and their personality after aswering a set of questions.
